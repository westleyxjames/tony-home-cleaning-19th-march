## 🚀 How to Use

### Local Development:
1. Simply open any `.html` file in a web browser
2. No build process or server required
3. All pages are fully functional

### Deployment:
Upload all files to any static hosting:
- **GitHub Pages**: Just push to gh-pages branch
- **Netlify**: Drag and drop the `/public` folder
- **Vercel**: Deploy as static site
- **Traditional hosting**: FTP upload all files

### Customization:

#### Change Colors:
Edit CSS variables in `styles/main.css`:
```css
:root {
    --color-blue-600: #2563eb;
    --color-green-600: #16a34a;
    /* etc. */
}
```

#### Update Content:
Edit HTML files directly - all content is in plain HTML

#### Modify Forms:
Form handlers are in `scripts/contact.js` and `scripts/main.js`
To connect to a backend, update the form submission functions
